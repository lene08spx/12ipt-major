Vector Notation

Unit Vectors

A vector with length 1 is a unit vector.

When \(\utilde{a}\ne\utilde{0}\), the vector \(\hat{\utilde{a}} = \frac{\utilde{a}}{|\utilde{a}|}\) is a unit vector in the same direction as \(\utilde{a}\).
(Multiplying by -1 will reverse the direction, pointing the vector in the opposite direction but with the same magnitude.)

Component Vectors

Using \(\utilde{i}\) to denote a horizontal unit vector.
Using \(\utilde{j}\) to denote a vertical unit vector.

Hence a vector \(x\utilde{i}+y\utilde{j}\) represents a line with gradient \(\frac{y}{x}\), and magnitude \(\sqrt{x^2+y^2}\).

Column Vectors

Instead of using \(\utilde{i}\) and \(\utilde{j}\), we can write the parts of a vector in brackets.

The component vector \(x\utilde{i}+y\utilde{j}\), is equivelant to the column vector \("\begin{bmatrix}x\\y\end{bmatrix}\)
